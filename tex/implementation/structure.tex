\section{Struktura projektu}
Struktura projektu zobrazená na diagramu \ref{project-structure} obsahuje soubory týkající se celého projektu,  a dále funkční celky nazvané \textbf{aplikace}. Projekt využívá více aplikací, ale v~rámci diplomové práce byla implementována jedna -- aplikace \texttt{web}, která je určena jak pro webovou aplikaci, tak pro implementaci API. Struktura aplikace \texttt{web} je zobrazena na diagramu \ref{structure-web}.
